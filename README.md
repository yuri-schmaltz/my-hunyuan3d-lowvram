# Hunyuan3D-2

A pinokio script for https://github.com/deepbeepmeep/Hunyuan3D-2GP


## Profiles
This wrapper includes optimized memory profiles for various hardware configurations:
- **Original**: Standard Hunyuan3D-2 execution.
- **Turbo**: Faster generation (requires more VRAM/compute).
- **MV**: Multi-view generation.
- **LowVRAM**: Specific profiles for 6GB/12GB VRAM cards using system RAM offloading (up to 48GB System RAM supported).

## Updates
The update script (`update.js`) safely pulls the latest changes and updates dependencies without deleting your existing virtual environment.
